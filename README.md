# 🌦️ EzForecast

EzForecast is a simple and clean weather application built as an assignment for the  
[Roadmap.sh Weather App Project](https://roadmap.sh/projects/weather-app).  
The project focuses on API usage, DOM updates, and basic UI interactions.

<a href="https://ezforecast.vercel.app/" target="_blank" rel="noopener noreferrer">
  ✅ Live Demo (Vercel)
</a>

---

## ✨ Features

- 🔍 **Search weather by city**
- 📍 **Use your current location**
- 🌡️ **Current temperature & condition**
- 💨 **Wind speed**
- 💧 **Humidity**
- 🕒 **Local time per city**
- 📅 **5-day forecast**
- 🎛️ **Animated search panel**
- 📱 **Fully responsive layout**
- ☁️ **Weather icons from WeatherAPI CDN**

---

## 🛠️ Tech Stack

- ⚡ **TypeScript**
- 🚀 **Vite**
- 🎨 **Tailwind CSS**
- 🌍 **WeatherAPI**
